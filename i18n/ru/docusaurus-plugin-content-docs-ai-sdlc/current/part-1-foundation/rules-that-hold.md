# Правила, которые работают

:::note[Урок в работе]
Этот урок ещё в работе. Ниже — что он охватит.
:::

Правило, которое агента просят соблюдать, — это совет. Правило, которое он не может обойти, — это контроль. Урок о том, почему исполняемые правила — правила как код — лучше прозы.
