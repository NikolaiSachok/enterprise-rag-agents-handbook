# Многоуровневый контроль

:::note[Урок в работе]
Этот урок ещё в работе. Ниже — что он охватит.
:::

Средства контроля подбирают так, чтобы механизмы были разными. Разные механизмы контроля выявляют разные классы дефектов. Урок о том, почему одного механизма контроля всегда мало.
