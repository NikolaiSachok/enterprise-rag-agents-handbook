# Artefakty ako jediné rozhranie

:::note[Pripravovaná lekcia]
Táto lekcia sa pripravuje. Nižšie je, čo pokryje.
:::

Etapy si prácu odovzdávajú cez písomné artefakty, nie cez rozhovor. To, čo prekročí hranicu, je zmluva medzi etapami.
