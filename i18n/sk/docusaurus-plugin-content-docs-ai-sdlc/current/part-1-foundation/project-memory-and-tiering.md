# Pamäť projektu a vrstvenie

:::note[Pripravovaná lekcia]
Táto lekcia sa pripravuje. Nižšie je, čo pokryje.
:::

Agent medzi behmi zabudne všetko. Riešením je trvalá pamäť projektu rozvrstvená podľa dôležitosti. Jej preplnenie však niečo stojí pri každom kroku, preto sa lekcia venuje aj tomu, čo vynechať.
