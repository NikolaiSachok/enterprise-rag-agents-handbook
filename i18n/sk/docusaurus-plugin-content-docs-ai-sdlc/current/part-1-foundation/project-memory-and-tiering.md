# Pamäť projektu a vrstvenie

:::note[Pripravovaná lekcia]
Táto lekcia sa pripravuje. Nižšie je, čo pokryje.
:::

Agent medzi behmi zabudne všetko. Liekom je trvalá pamäť projektu, rozvrstvená podľa dôležitosti. Preplniť ju je však priebežný náklad v každom kroku — lekcia preto rieši aj to, čo vynechať.
