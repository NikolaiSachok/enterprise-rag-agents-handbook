# Ako čítať dôkazy odboru

:::note[Pripravovaná lekcia]
Táto lekcia sa pripravuje. Nižšie je, čo pokryje.
:::

Ako odlíšiť meranie od marketingu. Každé tvrdenie dostane stupeň dôkazu — `MEASURED`, `REPORTED` alebo `ASSERTED`. Ukážeme aj, ako odhaliť zlý menovateľ či konflikt záujmov a dôjsť k pôvodnému zdroju.
