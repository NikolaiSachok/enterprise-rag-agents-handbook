# Detekcia verzus oprava: hranie na metriku

:::note[Pripravovaná lekcia]
Táto lekcia sa pripravuje. Nižšie je, čo pokryje.
:::

Oddeliť nájdenie chyby od jej opravy. Agent optimalizuje presne to, čo kontroluješ. Preto pozor na `reward hacking` — trik, ktorý bránu „ozelení" bez vykonania práce.
