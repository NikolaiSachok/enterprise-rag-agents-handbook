# Detekcia verzus oprava: obchádzanie metriky

:::note[Pripravovaná lekcia]
Táto lekcia sa pripravuje. Nižšie je, čo pokryje.
:::

Oddeliť nájdenie chyby od jej opravy. Agent optimalizuje presne to, čo kontroluješ. Preto si daj pozor na `reward hacking` — trik, ktorý bránu „ozelení" bez toho, aby sa práca skutočne vykonala.
